Éste es mi sitio personal. Actualmente me encuentro en proceso de transferirlo a GitHub.

Herramientas

La mayoría del trabajo lo realizo en la distribución Ubuntu del sistema operativo GNU/Linux. Mis principales herramientas son:

  • El documento se produce originalmente en markdown, el cual es un lenguaje de marcado muy sencillo,
  • El documento se convierte a HTML por medio de pandoc, y
  • El formato del documento producido se establece por medio de normalize.css y se ajusta para seguir las convenciones conocidas como solarized.
  • Para salvar a GitHub los archivos, utilizo el comando git(1).

Otras herramientas usadas como apoyo son:

  • El editor GNU Emacs,
  • El gestor de dependencias GNU Make, y
  • El procesador de macros GNU M4.
